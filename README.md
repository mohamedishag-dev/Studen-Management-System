# Student Management System

A simple Student Management System built with C# Windows Forms and File Handling (TXT Database).

## Features

* ✅ Admin Login System
* ✅Add new students
* ✅View all students
* ✅Search students by ID
* ✅Update student information
* ✅Delete students
* ✅Generate automatic Student IDs
* ✅Store data in a TXT file
* ✅Display students in DataGridView

## Technologies Used

* C#
* Windows Forms
* .NET Framework
* File Handling (TXT Database)

## Student Information

| Field      | Description          |
| ---------- | -------------------- |
| ID         | Unique Student ID    |
| First Name | Student First Name   |
| Last Name  | Student Last Name    |
| Age        | Student Age          |
| Gender     | Male / Female        |
| Phone      | Student Phone Number |
| Address    | Student Address      |


## Project Structure

* `clsStudent.cs` : Student class and file operations.
* `frmMain.cs` : Main form.
* `frmLogin.cs` : login screen form.
* `Database.txt` : Stores students data.

## Database Format

Example:

```text
STU001#//#Mohamed#//#Ishag#//#20#//#01000000000#//#Male#//#Cairo
```
